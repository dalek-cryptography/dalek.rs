dalek

dalek is a suite of cryptographic libraries and protocol implementations, written in the systems programming language Rust, for creating blazingly-fast, production-quality cryptographic applications.

Learn more








curve25519-dalek

A low-level cryptographic library for point, group, field, and scalar operations on a curve isomorphic to the twisted Edwards curve defined by -x²+y² = 1 - 121665/121666 x²y² over GF(2²⁵⁵ - 19).

ed25519-dalek

A memory-safe, fast, and efficient EdDSA signing and signature verification implementation which supports a well-designed API and friendly error management.

About


Our curve implementation was created by cryptographers, for cryptographers. Whether you're a seasoned cryptographic protocol implementer grounded in industry or a grad student prototyping your latest designs, dalek should be safe, easy, and friendly to use, allowing for rapid turnarounds from maths on a chalkboard to production-quality cryptographic implementations.

Start prototyping with curve25519-dalek!

Simply add the following to your Cargo.toml to get started: 

            [dependencies.curve25519-dalek]
            version = "1.0"
